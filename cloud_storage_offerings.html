Cloud Storage Offerings from AWS

Amazon offers multiple types of cloud storage solutions for different use cases.
Here we will look at Simple Storage Service (S3), Elastic File System (EFS), and
Elastic Block Store (EBS).

S3

S3 is an object storage solution that can store different types of files. It's a static storage solution used for web hosting, version management, and big data analytics. It is used through a web-interface and stores its files in buckets.

S3's key benefits are that it is the only object storage that allows you to block public access at the bucket or account level. It can also improve query performance by retrieving subsets of data instead of the entire object.

Elastic File System

EFS is a network file system that can be shared across different EC2 instances. It performs well for big data analytics, media processing, and content management. Its primary use case is for clustered databases and document sharing.

EFS's key benefits are that there is no need to worry about managing storage, updating hardware, or performing back-ups. It also provides data encryption at both the resting and transit phases making your data entirely secure.

Elastic Block Store

EBS is a high-performance block storage service designed to be used with EC2 for both throughput and transaction intensive workloads. It can handle relational and non-relational databases, big data analytics engines, and other workflows.

EBS's key benefits are that it functions very well when the load is very high. It has different volumes to choose from to optimize price points for any company. Lastly, with its elasticity you can increases storage without disruption to workload.
